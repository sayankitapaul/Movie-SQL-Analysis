# Movie SQL Analysis

This project analyzes movie data using PostgreSQL. It includes database creation, data insertion, and analytical SQL queries to explore movie ratings, revenue, genres, directors, and release years.

SQL concepts demonstrated:

* SELECT statements
* WHERE clauses
* GROUP BY and ORDER BY
* Aggregate functions (SUM, AVG, COUNT)
* Window functions (RANK, DENSE_RANK, ROW_NUMBER)
* Common Table Expressions (CTEs)
* Views
* Subqueries

The project showcases practical SQL skills used in data analysis and database management while providing insights into movie performance and trends.
